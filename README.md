# consignment
탁송/대리 자동매칭프로그램
### 1. 탁송차량과 기사의 거리를 검색하여 기사의 신뢰도를 구하고 높은 순으로 자동 매칭해준다
- 탁송차의 원활한 수송과 기사의 운행에 안전하고 신속하게 유지하도록하며 사후관리를 하여 효율을 높이고자 한다.
- 기사의 목록(인덱스,대리점,이름,연락처,생년월일,주소,면허종류,흡연유무,가입보험,사고점수,대리점평점,고객평점,신뢰도점수,신뢰도등급,비고)
- 개별기사의 건별 탁송 수행 목록(인덱스,차모델,출발주소,도착주소,수행완료,특이사항,비고)
 
### 2. 탁송전후 사진을 비교하여 사고(파손,스크래치)를 비교 분석하여 단계별 점수를 부여한다
- 중고차의 상태를 확인하게 하여 고객과 기사의 마찰을 최소화하고 고객에게 신뢰도를 주게하며, 기사의 무과실 입증에도 도움을 주고, 사고파손을 곧바로 확인하여 관리함으로 대응을 신속하게 한다.

- 모델 선택: 단순한 사각형 박스 탐지(Object Detection)보다는 파손 부위의 정확한 형태를 따내는 인스턴스 분할(Instance Segmentation) 모델인 YOLOv8-seg 또는 YOLOv9 이상을 추천합니다. Ultralytics 공식 문서에서 최신 모델 사양을 확인할 수 있습니다.
- 데이터셋 확보: 좋은 탐지 프로그램을 만들려면 실제 사고 차량의 스크래치, 유리 찍힘 사진 데이터가 필요합니다. AI Hub의 자동차 파손 이미지 데이터셋 등을 활용해 모델을 추가 학습(Fine-tuning)시켜야 정확도가 올라갑니다.
- 유리 찍힘 특화: 유리 찍힘은 미세하므로 고해상도 이미지가 필요하며, Canny Edge Detection 같은 전통적인 영상 처리 기법을 결합하여 균열의 패턴을 분석하면 더욱 효과적입니다.
- 고도화 방안 (2026년 기준)
실시간 영상 처리: 스마트폰 카메라로 차량을 한 바퀴 돌면 실시간으로 파손을 체크하는 앱 형태로 발전시킬 수 있습니다.
. 심도 카메라(Lidar/Depth) 활용: 아이폰의 LiDAR 센서 등을 이용해 찌그러진 깊이(Depth)를 측정하여 수리비를 산출하는 기능을 추가할 수 있습니다.
  
### 3. 탁송전후 고객(대리점)의 만족도(리뷰)에 따라 점수를 부여한다.
- 탁송전후 고객과의 친절한 응대를 함으로 고객과 기사의 관계개선에 기여하고 안전운행에 기여한다.
- 탁송 물건의 공급을 늘리고 탁송기사에게 배차의 기회를 제공하여 수익을 증대할 수 있는 길이 된다.
  
### 4. 탁송운행 구간의 교통혼잡도, 기상상태, 최적의 운행노선, 특이사항을 자동으로 수집하여 수행기사에게 전달해준다
- 운행기사에게 운행노선의 최소의 정보를 제공함으로 배차받은 운행의 안전하고 신속한 완료를 하게 한다.
- 도착 탁송기사의 연결 운행배차를 도착전에 알려주거나 가능배차 정보를 제공 또는 배차가 없는 늦은 저녁시간에 대리/탁송을 연결해주거나 이동 차편(타기사 탁송노선 참고)을 제공해주는 방안도 고려하며, 주변 숙소 정보를 제공해주는 서비스도 기사의 운행과 연결된 향상 서비스이다.
