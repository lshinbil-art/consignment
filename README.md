# consignment
탁송/대리 자동매칭프로그램
### 1. 탁송차량과 기사의 거리를 검색하여 기사의 신뢰도를 구하고 높은 순으로 자동 매칭해준다
- 탁송차의 원활한 수송과 기사의 운행에 안전하고 신속하게 유지하도록하며 사후관리를 하여 효율을 높이고자 한다.
- 기사의 목록(인덱스,대리점,이름,연락처,생년월일,주소,면허종류,흡연유무,가입보험,사고점수,대리점평점,고객평점,신뢰도점수,신뢰도등급,비고)
- 개별기사의 건별 탁송 수행 목록(인덱스,차모델,출발주소,도착주소,수행완료,특이사항,비고)
 
### 2. 탁송전후 사진을 비교하여 사고(파손,스크래치)를 비교 분석하여 단계별 점수를 부여한다
- 중고차의 상태를 확인하게 하여 고객과 기사의 마찰을 최소화하고 고객에게 신뢰도를 주게하며, 기사의 무과실 입증에도 도움을 주고, 사고파손을 곧바로 확인하여 관리함으로 대응을 신속하게 한다.
  
### 3. 탁송전후 고객(대리점)의 만족도(리뷰)에 따라 점수를 부여한다.
- 탁송전후 고객과의 친절한 응대를 함으로 고객과 기사의 관계개선에 기여하고 안전운행에 기여한다.
  
### 4. 탁송운행 구간의 교통혼잡도, 기상상태, 최적의 운행노선, 특이사항을 자동으로 수집하여 수행기사에게 전달해준다
- 운행기사에게 운행노선의 최소의 정보를 제공함으로 배차받은 운행의 안전하고 신속한 완료를 하게 한다.
